# UniMoyenne Tiaret

Application Flutter pour calculer les moyennes des étudiants (IAD, GL, GI, RT).

Fonctionnalités :
- Calcul moyenne par semestre
- Gestion TD / TP / Examen
- Décision Admis / Ajourné
- Support M1 / M2
- Plusieurs spécialités
