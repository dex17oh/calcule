# University GPA Calculator (Flutter)

University GPA Calculator is a modern Flutter mobile application that helps university students calculate their semester GPA easily based on modules, coefficients, TD, TP, and exams.

Features:
- Splash Screen with animation
- Specialities: IAD (Artificial Intelligence & Digitalisation), GL (Software Engineering), GI (Computer Engineering), RT (Networks & Telecommunications)
- Year & Semester selection (M1 / M2 – S1 to S4)
- GPA calculation with real coefficients
- Water Drop Bottom Navigation Bar
Q- Profile page
- Custom reusable AppBar
- Clean and responsive UI

How to run the project:
1. Install Flutter from https://docs.flutter.dev/get-started/install
2. Check Flutter installation by running: flutter doctor
3. Clone the project using: git clone https://github.com/your-username/university-gpa-calculator.git
4. Open project folder: cd university-gpa-calculator
5. Install dependencies: flutter pub get
6. Run the application: flutter run

Project structure:
lib/data contains all programmes and modules
lib/logic contains GPA calculation logic
lib/models contains data models
lib/pages contains application pages
lib/widgets contains custom widgets like AppBar and BottomNavigationBar
lib/themes contains colors and theme configuration
main.dart is the entry point of the application
